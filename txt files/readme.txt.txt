In room 1, the door opens with two keys and the switch makes a wall disappear.
In room 2, the door opens with one key and the switch is on.
In room 3, the door opens with three keys and the switch is on.

The score system works as follows:
- Correct riddle = 50 points
- Incorrect riddle = -30 points and return to start
- Finish room with full health for both players = 500 points
- Finish room before clock timeout = 200 bonus points
- Finish room after clock reaches zero = 50 points

The timeout displayed on screen shows remaining time for bonus score.
When using torch, remaining time is hidden to increase game difficulty.
